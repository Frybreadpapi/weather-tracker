# weather-tracker
## Description

What was your motivation? My motivation was to create a weather tracker for not only myself but for other people that want to support my work.
Why did you build this project? I built this project to help people find out what type of weather they are going to experience not only for the present day but future days!
What problem does this solve? It solves problems that people might face when they are traveling, getting ready for the day, etc.
